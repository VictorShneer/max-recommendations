Создание API-ключа в Яндекс-метрике

Для того, чтобы платформа аналитики смогла получить доступ к данным вашего счетчика Яндекс-метрики, нужно получить API-ключ, который вам нужно будет указать при создании интеграции. Ниже предоставляем подробную инструкцию по получению API-ключа

  1. В первую очередь вам нужно зайти на страницу вашего счетчика, чтобы убедиться, что вы залогинены в системе и имеете доступ к нему.

  2. Далее заходим на страницу https://oauth.yandex.ru/client/new и создаем новое приложение.

    Вводим любое название приложения, в разделе "платформы" выбираем "Веб-сервисы". Нажимаем на ссылку "Подставить URL для разработки".

    В разделе "Доступы" выбираем Яндекс.Метрика и отмечаем пункт "Получение статистики, чтение параметров своих и доверенных счетчиков"

    Нажимаем "Создать приложение" в самом низу страницы.

  3. Видим страницу с данными, необходимыми для получения ключа. ВНИМАНИЕ - страницу пока не закрываем!!!

  4. Формируем ссылку вида:

    https://oauth.yandex.ru/authorize?response_type=token&client_id=<идентификатор приложения>

    Здесь <идентификатор приложения> - это ID из предыдущего пункта.

  5. Заходим по ссылке, сформированной в предыдущем пункте и видим примерно такую страницу:

    Код, который вы видите в центре страницы - это и есть необходимый нам API-ключ. Вам нужно его скопировать (лучше сохранить куда-нибудь, чтобы в будущем не проходить процедуру получения ключа заново) и указать при создании интеграции.
